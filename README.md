## Overview  

**AgroLink** is a full-stack web platform designed to bridge the gap between farmers and buyers.  
It enables transparent trade, real-time communication, and access to essential resources â€” all on a single portal.  

The system is built using **PHP, MySQL, HTML, CSS, JavaScript, Bootstrap, and jQuery**, ensuring a dynamic and responsive interface for all stakeholders.

---

## Key Features  

-  **Farmer Dashboard:** List crops, view buyers, manage prices.  
-  **Buyer Dashboard:** Browse, negotiate, and purchase directly.  
-  **Weather Updates:** Real-time weather insights using API integration.  
-  **Government Schemes:** Display latest farming-related initiatives.  
-  **Farming Tutorials:** Knowledge hub for sustainable practices.  
-  **Community Forum:** Farmers can connect, discuss, and share updates.  
-  **Admin Panel:** Manage users, products, and reports.
